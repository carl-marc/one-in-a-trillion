# v17.3.0 (1084)

## Features
- Multiplayer Battles
  - Added daily 1vs1's
  - Added weekly 1vs1's
  - Multiple 1vs1 battles can be played at the same time (1 quick or epic + 1 daily + 1 week)
  - Most taps bonus percentages changed:
    - Quick: 30%
    - Epic: 20%
    - Day: 10%
    - Week: 5%
